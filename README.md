# Detection and Classification of Online Threat Messages on X (Twitter) Using Machine Learning and NLP

## Overview
This project focuses on detecting and classifying online threat messages on X (Twitter) using Machine Learning and Natural Language Processing (NLP) techniques. The objective is to develop an automated system capable of identifying harmful or threatening content in social media text.

## Objectives
- Detect online threat messages from social media posts
- Classify threat-related content using ML models
- Apply NLP techniques for Thai/English text preprocessing
- Evaluate model performance using standard metrics

## Dataset
The dataset consists of collected messages from X (Twitter), including labeled threat and non-threat messages.

### Data Preprocessing
- Text cleaning
- Tokenization
- Stopword removal
- TF-IDF feature extraction
- Label encoding

## Machine Learning Models
The following models were implemented:
- Naive Bayes
- Support Vector Machine (SVM)
- Random Forest
- Logistic Regression
